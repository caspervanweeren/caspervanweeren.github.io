het geheugen

Het geheugen van een apparaat behoort tot de fysieke laag. Het geheugen is opgedeeld in 3 verschillende componenten, namelijk ROM, RAM en het permanent geheugen. Op deze pagina kan je uitleg vinden over hoe deze drie componenten in elkaar zit, wat de taken zijn van de verschillende soorten gehugen en wat het verschil is tussen de verschillende vormen van geheugen

RAM

RAM staat voor random-acces memory, dit betekent geheugen met willekeurige toegang. RAM is een essentieel onderdeel in de computer, het wordt ook wel het werkgeheugemn genoemd. Het wordt in allerlei apparaten gebruikt, zoals laptops, Macbook, desktops en NAS. Het zorgt voor de tijdelijke opslag van programma's. Via het werkgeheugen voert de pc de taken uit die door de processor verstuurd wordt. Zo kan het er bijvoorbeeld voor zorgen dat je meerdere tabladen tegelijk open hebt staan. Hieronder kan je een foto bekijken van hoe het RAM geheugen er in het echt uit ziet.

ROM

ROM staat voor read-only memory. Een read-only memory is een geheugenopslagmedium in elektronische apparaten. Dit geheugen kan alleen gelezen worden. Het bevat gegevens die er voor zorgen dat een computer kan opstarten en goed werkt, zoals de UEFI instellingen. Hierin staan instelling waarmee testen worden uitgevoerd voordat de computer de rest van de data laad. Het verschil tussen ROM en RAM is dat RAM tijdelijk is en dat ROM permanent is. Bij ROM wordt het geheugen dus bewaard als je je apparaat afsluit in tegenstelling tot bij het RAM geheugen
