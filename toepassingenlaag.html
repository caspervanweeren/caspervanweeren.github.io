    toepassingen laag

    Wat is de toepassingenlaag?

    De toepassingenlaag is de laag uit het drielagenmodel die communicatie protecollen en koppelvlakken(ook wel interfaces genoemd) specificeert. Deze laag maakt het mogelijk om computers met elkaar te laten communiceren via telecommunicatienetwerken en computernetwerken. Volgens het OSI-model is de toepassingenlaag de laag die de ontvangen informatie aanbiedt aanbiedt aan de gebruikersapplicatie.

    Apps

    API's

    API staat voor Application Programming Interface is op een eenvoudige manier uitgelegd een standaardkoppeling tussen computersystemen zodat ze eenvoudig gegevens kunnen uitwisselen en samenwerken. Een API wordt gebruikt voor de comminucatie tussen de logischelaag en de toepassingenlaag. Ook zorgt een API ervoor hoe bepaalde knoppen tijdens het gebruik van een app op het scherm komen.
